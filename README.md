# ai-governor

**Model-level governance and compliance framework for LLM systems**

ai-governor provides a runtime control layer between applications and
Large Language Models (LLMs), enabling enforceable policies, auditability,
and compliance-by-design.

## Why this exists
See [docs/WHY_THIS_EXISTS.md](docs/WHY_THIS_EXISTS.md)

## Project status
Early-stage / experimental  
APIs will change. Contributions welcome.

## Goals
- Policy-as-code for LLM systems
- Model-agnostic governance layer
- Runtime enforcement (pre / post inference)
- Audit-grade evidence generation

## Non-goals
- Legal compliance certification
- Prompt-only safety filtering
- Hosted SaaS platform
